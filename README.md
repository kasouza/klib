# klib

A c library for stuff I need in different projects.

Right now it has:
  - Data structures:
    - Linked List
   

What I plan on adding:
  - Data structures
    - Hashmap
    - Maps
    - Dynamic array

  - Vectors and matrices


# Compile
Compile it with
```
mkdir build
cd build
cmake .. && cmake --build
```

Then link it to your project.

It's all posible to simple add as a library directly to you CMake project with target_link_libraries.
